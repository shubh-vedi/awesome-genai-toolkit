## 📖 What Is This?

This is an **AI agent skill** that generates branded meme creatives in the trending **"Baccha hai tu mera"** format — the viral Indian meme where a warm, fatherly cartoon character (Jameel Mamu) promotes products with affectionate Hindi dialogue.

### What You Get

| Output | Description |
|--------|-------------|
| 🎨 **AI Image Prompt** | Ready-to-use prompt for Midjourney / DALL-E / Ideogram |
| ✍️ **Text Overlay** | Hindi-English meme copy with color/style guides |
| 📱 **Instagram Caption** | Hinglish caption with hashtags & CTA |
| 🎬 **Reel Script** | Optional storyboard with timing & audio cues |

---

## 🚀 Quick Start

### 1. Give the agent a brand + product

```
Create a meme for Chai Point's Masala Chai
```

### 2. Receive all four outputs

The skill generates an AI image prompt, text overlay copy, Instagram caption, and (optionally) a reel storyboard — all in the Jameel Mamu style.

### 3. Generate the image

Paste the AI image prompt into **Midjourney**, **DALL-E**, **Ideogram**, or any image generator, then add the text overlay using Canva, Figma, or any editor.

## 🎨 The Visual Style

| Element | Details |
|---------|---------|
| **Character** | Warm, middle-aged Indian man — slicked hair, round glasses, mustache, teal Nehru jacket |
| **Art style** | Cartoon caricature, halftone textures, cream/beige background |
| **Pose** | Holding/presenting product toward viewer |
| **Typography** | "Baccha" (large red) + "hai tu mera" (brown/gold) + product pitch |
| **Aspect ratio** | 4:5 (Instagram-optimized) |
