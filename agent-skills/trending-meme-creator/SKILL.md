# 🔥 Trending Meme Creator Skill

This skill generates branded meme creatives in the viral **"Baccha hai tu mera"** (Jameel Mamu) style — the trending Indian meme format where a warm, fatherly cartoon character promotes products with affectionate Hindi dialogue.

---

## 🎨 The Meme Format

### Visual Style

- **Character**: A warm, middle-aged Indian man (uncle/mamu archetype) — slicked-back hair, round glasses, mustache, wearing a dark blue/teal Nehru jacket or bandhgala. Gentle, smiling expression. **Cartoon/caricature illustration** style, NOT photorealistic.
- **Art style**: Digital caricature with warm halftone/comic textures, slightly exaggerated features, soft paper-like background (cream/beige), hand-drawn feel
- **Pose**: Character always holding/presenting the product toward the viewer with a proud, offering gesture
- **Product placement**: Product prominently displayed in character's hands with brand logo clearly visible
- **Typography**: Mixed Hindi-English text in a specific layout:
  - "Baccha" in large bold red/orange brush script (top)
  - "hai tu mera" in brown/gold handwritten style
  - Product pitch text alternating between red and brown/gold
  - Text flows vertically down the right side of the image
- **Color palette**: Warm cream background, red + brown/gold text, teal jacket on character, product colors pop against the muted background

### Copy Formula

The dialogue follows this exact pattern:

```
Baccha hai tu mera...
[Ye le / Le] [Brand Name] ka [Product Name] [action verb]!
```

**Action verbs by category:**

| Category | Verbs |
|----------|-------|
| Food/Drinks | "khaa le", "pi le", "try kar", "chakh le" |
| Music/Entertainment | "sun le", "dekh le" |
| Apps/Tech | "download kar le", "try kar le", "use kar le" |
| Fashion/Products | "pehen le", "le le", "try kar le" |
| Services | "book kar le", "try kar le" |

## 🚀 How to Use This Skill

When a user provides a brand/product, generate the following outputs:

### Output 1: AI Image Generation Prompt

Write a detailed prompt (for Midjourney, DALL-E, Ideogram, or any AI image generator) that recreates this exact style.

**Prompt Template:**

```
Digital caricature illustration of a warm, smiling middle-aged Indian man with slicked-back
dark hair, round metal glasses, a thick mustache, wearing a dark teal Nehru jacket. He is
holding out [PRODUCT DESCRIPTION WITH BRAND DETAILS] toward the viewer with a proud,
offering gesture. The product has [BRAND LOGO/DETAILS] clearly visible. Style: warm cartoon
caricature with halftone dot texture, slightly exaggerated features, soft cream/beige
paper-textured background. Composition: character on the left side, large empty space on the
right for text overlay. Warm, inviting, fatherly mood. No text in the image.
Aspect ratio: 4:5 (portrait, Instagram-optimized).
```

### Output 2: Text Overlay Copy (Hindi-English Mix)

Generate the meme copy in this format:

| Line | Style | Content |
|------|-------|---------|
| 1 | Large, Red/Orange | Baccha |
| 2 | Brown/Gold | hai tu mera |
| 3 | Brown/Gold | Ye le |
| 4 | Red/Brown | [Brand Name] ka |
| 5 | Large, Red | [Product Name] |
| 6 | Brown/Gold | [action verb]! |

### Output 3: Instagram Reel/Post Caption

Write a caption for Instagram in Hinglish (Hindi-English mix) that:

- References the "Baccha hai tu mera" trend
- Tags the brand
- Uses relevant hashtags
- Includes a CTA
- Keeps the warm, playful Jameel Mamu tone

**Caption Template:**

```
Baccha hai tu mera... 🤗

[Witty line about the product using the mamu/uncle voice]

[Brand Name] ke saath [benefit/value proposition] — Jameel Mamu approved! 😎

[CTA — "Try karo aaj hi!" / "Order karo abhi!" etc.]

#BacchaHaiTuMera #JameelMamu #[BrandHashtag] #TrendingMeme #BrandCollab #[CategoryHashtag]
```

### Output 4: Reel Script / Storyboard (Optional — generate if user asks for reels)

| Segment | Time | Content |
|---------|------|---------|
| Hook | 0–3s | Zoom into the meme image with dramatic sound effect |
| Reveal | 3–6s | Full meme with voiceover reading dialogue in warm uncle voice |
| Product Shot | 6–10s | Transition to real product footage/photo |
| CTA | 10–15s | Brand logo + "Order now" / "Visit us" text |
| Audio | — | Trending audio clip or original Jameel Mamu dialogue |

## 🎭 Variations & Advanced Use Cases

### Multi-Product Series

Create a series where each meme features a different product but maintains the same character and format — builds a recognizable campaign.

### Seasonal Adaptations

| Season | Ideas |
|--------|-------|
| ☀️ Summer | Cold drinks, ice cream, AC remote |
| 🎉 Festivals | Festive attire, gifts, sweets |
| 🏏 Cricket season | Cricket gear + product |
| 🥭 Mango season | Hapus/Alphonso mango products (the OG!) |

### Category-Specific Adjustments

| Category | Visual Tweak |
|----------|-------------|
| Food & Beverage | Character takes a sip/bite, product has steam/freshness cues |
| Tech/Apps | Character holds phone showing the app, or the product box |
| Fashion | Character wears or presents the clothing item |
| Entertainment | Character holds album art, movie poster, or book |
| Services | Character holds a booking card or gift voucher |

## ✅ Quality Checklist

Before delivering, verify:

- [ ] AI prompt specifies: cartoon caricature style (NOT photorealistic)
- [ ] AI prompt includes: halftone texture, cream background, teal jacket
- [ ] AI prompt leaves text space on the right side
- [ ] AI prompt specifies 4:5 aspect ratio for Instagram
- [ ] Copy follows the exact "Baccha hai tu mera" + product pitch formula
- [ ] Action verb matches the product category
- [ ] Brand name and product name are correctly placed
- [ ] Caption is Hinglish with appropriate hashtags
