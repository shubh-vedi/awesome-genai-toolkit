# 🤖 Smart Agents

> Autonomous AI agents that reason, plan, and execute tasks — from beginner-friendly bots to advanced multi-agent orchestration.

## 📂 Structure

```
agent-skills/
├── starter/          # Simple single-agent apps (great for beginners)
├── advanced/         # Complex agents with tool use, planning & memory
└── multi-agent/      # Teams of agents collaborating on tasks
```

## 🔥 Why This Is Trending

AI Agents are the #1 trend in GenAI — moving beyond simple chat to autonomous systems that can browse the web, write code, manage finances, conduct research, and more. Frameworks like CrewAI, LangGraph, OpenAI Agents SDK, and Google ADK are making agent development accessible to every developer.

## 🚀 Getting Started

```bash
pip install -r requirements.txt
python main.py
```
