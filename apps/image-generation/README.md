# 🎨 AI Image Generator

Generate stunning images from text prompts using **DALL-E 3** with a **Streamlit** gallery interface.

![Python](https://img.shields.io/badge/Python-3.9+-blue)
![OpenAI](https://img.shields.io/badge/OpenAI-DALL--E_3-purple)
![Streamlit](https://img.shields.io/badge/Streamlit-1.40-red)

## Features
- 🎨 Generate images from text prompts
- 🖼️ Gallery view with prompt history
- 💡 Example prompts to get started
- 🎛️ Size, quality, and style controls

## Quick Start

```bash
cd apps/image-generation
pip install -r requirements.txt
streamlit run app.py
```

## Environment

Create `.env`:
```
OPENAI_API_KEY=sk-...
```
