# 🌐 Multi-Modal AI

Analyze images, compare visuals, extract text (OCR), read charts, and transcribe audio — all in one **Streamlit** app.

![Python](https://img.shields.io/badge/Python-3.9+-blue)
![OpenAI](https://img.shields.io/badge/OpenAI-GPT--4o-purple)
![Streamlit](https://img.shields.io/badge/Streamlit-1.40-red)

## Features
- 👁️ **Image Analysis** — Describe, identify objects, generate alt text
- 🔍 **Compare Images** — Side-by-side multi-image comparison
- 📝 **OCR & Charts** — Extract text, analyze graphs and charts
- 🎧 **Audio** — Transcribe + analyze audio content

## Quick Start

```bash
cd apps/multi-modal
pip install -r requirements.txt
streamlit run app.py
```
