## 🎯 Apps

Production-ready GenAI applications with **Streamlit UI**, full source code, and documentation.

```bash
# Run any app in seconds
cd apps/<app-name>
pip install -r requirements.txt
streamlit run app.py
```

| App | Description | Stack |
|-----|-------------|-------|
| [🔍 RAG Chatbot](apps/rag-chatbot/) | Chat with your documents using RAG | Streamlit, LangChain, ChromaDB |
| [🤖 AI Agent Team](apps/ai-agent/) | Multi-agent research & writing pipeline | Streamlit, CrewAI |
| [🎨 Image Generator](apps/image-generation/) | Generate images with gallery view | Streamlit, DALL-E 3 |
| [🎙️ Voice AI](apps/voice-ai/) | Voice chat, text chat + TTS studio | Streamlit, Whisper, TTS |
| [🌐 Multi-Modal](apps/multi-modal/) | Image analysis, OCR, comparison, audio | Streamlit, GPT-4o |

Each app includes a `README.md`, `requirements.txt`, and well-commented Streamlit code.
